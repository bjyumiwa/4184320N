# GitHub Pages 用お世話ゲーム

このフォルダは、そのまま GitHub リポジトリのルートに置いて公開できる構成です。

## 使い方

1. `index.html` をリポジトリのルートに置く
2. `assets/images` フォルダも同じ階層で置く
3. GitHub Pages を有効にする

## この版で入れてあるもの

- 3日間のお世話ゲーム
- マカロン選択から始まるキャラ選択
- いつものキャラクター画像の使用
- 行動ログの localStorage 保存
- CSV / JSON 出力
- 再生成後の appearance / personality / memory / moral 条件反映

## 画像ファイル

ファイル名に空白があった画像は GitHub Pages で扱いやすいように整えています。

- `purple_sleep.png`
- `space_background.png`
